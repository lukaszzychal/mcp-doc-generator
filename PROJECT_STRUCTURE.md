## ğŸ“ Opis katalogÃ³w

### `src/`
MCP server source code. Contains:
- `server.py` - main MCP server with tool definitions
- `tools/` - modules generating diagrams (PlantUML, Mermaid, Graphviz, draw.io)
- `templates/` - document templates (ADR, API Spec, C4, Microservices)
- `utils/` - helper utilities (file management, Polish support)

### `scripts/`
Helper scripts for local usage:
- `install.sh` - automatic project installation
- `mcp_client.py` - MCP client for generating diagrams from prompts
- `generate_examples.py` - script to generate all example files

### `tests/`
Tests and test files:
- Tests for all MCP tools
- Cursor integration tests
- Files generated during tests (in `tests/output/`)

### `docs/`
Project documentation:
- Usage guides
- Configuration instructions
- Test results
- Technical documentation

### `examples/`
Usage examples:
- Example prompts
- Example diagrams
- Usage patterns

### `output/`
Output directory for generated files.
**Note:** In Docker container it's mounted as `/app/output`.

## ğŸ”§ Usage

### Locally (without Cursor)
```bash

```

### With Cursor
```bash

```
